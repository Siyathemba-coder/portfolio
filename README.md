# Siyathemba Msimang | Personal Portfolio

This repository contains the source code for my personal portfolio website, built with **HTML, CSS, and JavaScript**. It showcases my projects, skills, education, experience, and contact information.

## Overview
The portfolio includes the following sections:

- **About Me**: Introduction and background.
- **Education**: My current academic pursuit at the University of KwaZulu-Natal.
- **Skills**: Programming languages, tools, and technical skills.
- **Experience**: Hands-on project and academic experience.
- **Projects**: Detailed descriptions of key projects including:
  - File Encryption System
  - Car Rental Management System
  - Network Scanner
- **Contact**: Email and phone contact details.

## Features
- Fully responsive layout.
- Smooth scrolling navigation.
- Clean and modern design.
- Easy to update and maintain.

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/siyathemba-coder/portfolio.git
   ```
2. Open `index.html` in your browser to view the portfolio locally.
3. Edit sections or add new projects by updating the HTML and CSS files.

## Deployment
This website can be deployed using:
- **GitHub Pages**: Free hosting for static websites.
- **Netlify / Vercel**: Free hosting with drag-and-drop deployment.
